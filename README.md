- Sistema simples de Gestão de Abrigo de Animais (MySQL).

- O projeto foi feito na plataforma MySQl Workbench e está em desenvolvimento.

- Este projeto foca em gerenciar o cadastro de animais quue chegam ao abrigo, no cadstro das pessoas que tem interesse de adotar, no histórico médico de todos os animais e também nas adoções que foram feitas ou rejeitadas.
